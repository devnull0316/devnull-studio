作品

GAME — 墨蛍
墨蛍 — すみほたる —
いちおしの一作

筆で描いて、蛍を星へ。

指先が筆になる。描いた墨の壁で跳ねる蛍を導き、時間が切れる前に光る星へ。描いた墨は消えない — やがて自分の墨が自分を阻み、一局ごとにあなたの軌跡が一幅の墨絵になる。

  • 夜巡り — 夜ごとに変わる地形を攻略するステージモード
  • 無限の夜 — 宵・夜半・暁、どこまで生き延びるか
  • 掛け軸 — 遊んだ跡がそのまま絵になり、保存・共有できる
いますぐ遊ぶ — 無料 iPad / Apple Pencil 推奨 · インストール不要
02

実験室

LAB — 配信を楽しくする道具の試作
🌱

コメントの庭

画面の隅に置く小さな庭。視聴者が「草」と打つと草が生え、「8888」で花火が上がり、初見さんは蛍になって住み着く。配信を邪魔しない常駐演出。

▶ ブラウザで実証デモを触れます
🏁

レース投票

コメントで「1」と打つと、その選択肢が走り出すアンケート。投票者の名前が刻まれ、接戦がそのまま配信の見せ場になる。

▶ ブラウザで実証デモを触れます
📋

配信カルテ(構想中)

チャットの流れから、盛り上がった瞬間・常連さんの動き・初見さんを1枚にまとめる配信後の日報。切り抜き候補も自動で。

準備中
※ 実験室のデモはチャット欄を内蔵しており、配信や特別な設定なしで動きを確認できます。すべてローカル処理で、コメントが外部に送信されることはありません。

スタジオについて

ABOUT
小さくても、
手触りのいいものを。

DevNull Studioは、東京を拠点にWebの企画・設計・開発までを一貫して行う独立系のスタジオです。アプリのような体験をインストールなしで届けられる、ブラウザという場所の可能性を信じています。

自社作品のゲームづくりで磨いた「触って気持ちいい」の感覚を、配信ツールやWebアプリケーションにも注いでいます。TypeScript / React / モバイルアプリ開発のご相談も承ります。
